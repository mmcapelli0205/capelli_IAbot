## 2. Skills / repositórios identificados nas capturas e resumos do vault

### MCP Server
- **Link:** https://github.com/modelcontextprotocol/servers
- **Como instalar/usar:** abrir o repositório e seguir o README do servidor que você quer usar.
- **O que faz:** liga a IA com ferramentas externas, como arquivo, banco, planilha, navegador e serviços.
- **Em português simples:** faz a IA parar de viver só no chat e começar a mexer com dados e sistemas reais.

### Marketing Skills
- **Link:** https://github.com/The-Pocket/Tutorial-Codebase-Knowledge/tree/main/Marketing%20Skills
- **Como instalar/usar:** clonar ou copiar a pasta e seguir o README.
- **O que faz:** ajuda a IA a pensar marketing, copy, oferta, páginas e conteúdo.
- **Em português simples:** serve pra vender melhor e escrever material comercial com mais direção.

### Brand Guideline
- **Link:** https://github.com/Ejb503/claude-code-subagent-collection/tree/main/brand-guideline
- **Como instalar/usar:** seguir o README da skill.
- **O que faz:** mantém padrão visual, voz da marca e consistência do material.
- **Em português simples:** evita cada tela, página ou texto sair com uma cara diferente.

### Claude SEO
- **Link:** https://github.com/AdemAkdag/claude-seo
- **Como instalar/usar:** seguir o README do projeto.
