## Repositórios / Skills

### 1. MCP Server
- **Link:** https://github.com/modelcontextprotocol/servers
- **Como instalar/usar:** veja os READMEs de cada servidor dentro do repositório
- **O que faz:** conecta a IA com ferramentas e dados de fora, tipo arquivos, banco, planilha e serviços externos.

### 2. Marketing Skills
- **Link:** https://github.com/The-Pocket/Tutorial-Codebase-Knowledge/tree/main/Marketing%20Skills
- **Como instalar/usar:** baixar/clonar o repositório e seguir o README da pasta
- **O que faz:** ajuda a IA a escrever melhor pra vender, fazer conteúdo, melhorar páginas e pensar em marketing.

### 3. Brand Guideline
- **Link:** https://github.com/Ejb503/claude-code-subagent-collection/tree/main/brand-guideline
- **Como instalar/usar:** clonar o repositório e usar a skill/pasta conforme o README
- **O que faz:** mantém a identidade visual e o jeito da marca consistentes.

### 4. Claude SEO
- **Link:** https://github.com/AdemAkdag/claude-seo
- **Como instalar/usar:** seguir o README do repositório
- **O que faz:** ajuda a criar páginas e textos pensando em aparecer melhor no Google.

### 5. Optimization / Context Engineering
- **Link:** https://github.com/Veraticus/awesome-claude-code-subagents
- **Como instalar/usar:** clonar o repositório e escolher os subagents/skills que fizerem sentido
- **O que faz:** organiza melhor o contexto da IA pra ela errar menos e trabalhar de forma mais eficiente.

### 6. Skill Creator
- **Link:** https://github.com/hesreallyhim/awesome-claude-code/tree/master/prompts/skill-creator
- **Como instalar/usar:** usar o prompt/skill do repositório como base
- **O que faz:** cria novas skills pra transformar tarefas repetidas em processo pronto.

### 7. Remotion
- **Link:** https://github.com/remotion-dev/remotion
- **Como instalar/usar:** seguir a documentação oficial do projeto
- **O que faz:** cria vídeos por código. Útil pra conteúdo, anúncios e vídeos automáticos.

### 8. File Search
- **Link:** https://github.com/ayanami2003/file-search-mcp
- **Como instalar/usar:** seguir o README do repositório
- **O que faz:** deixa a IA achar arquivos, trechos de código e partes importantes mais rápido.

### 9. NotebookLM
- **Link:** https://notebooklm.google/
- **Como instalar/usar:** usar via web
- **O que faz:** organiza conteúdo, resume material e ajuda a estudar ou juntar referência.

### 10. Web Artifacts
- **Link:** https://github.com/dwyle/claude-code-web-artifacts-mcp
- **Como instalar/usar:** seguir o README do repositório
- **O que faz:** ajuda a IA a montar interface web, páginas, blocos visuais e protótipos mais rápido.
